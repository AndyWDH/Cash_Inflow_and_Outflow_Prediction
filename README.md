# 资金流入流出预测
# Cash_Inflow_and_Outflow_Prediction
# 数据源从阿里天池下载，使用时间序列模型，计算获得周期因子，使用ARIMA工具来预测资金的流入流出
# 思路：
#  1、先绘出资金流入流出的走势图
#  2、截取2014-03-01到2014-08-31的数据，进行汇总，然后分别对 total_purchase_amt 和 total_redeem_amt 使用 ARIMA(7,1,7) 进行建模，然后对未来 20140901到20140930 这30天的申购和赎回的总金额进行预测
#  3、用周期因子来对未来30天，即 total_purchase_amt 和 total_redeem_amt进行预测，这里的周期因子可以是 weekday（即周一到周日的），也可以是 day（即1号到31号的），整理思路及置信度（0-100）
#  4、实现周期因子，用周期因子来做预测，周期因子 weekday 和 day 可以考虑用乘法关系
#  5、分析预测结果是否存在很大偏差，优化
# 最终结果：可以实现120分以上的结果
